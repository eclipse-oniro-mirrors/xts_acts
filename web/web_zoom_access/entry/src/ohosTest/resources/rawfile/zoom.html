Zoom Access Test Page

This page is used for testing the zoomAccess functionality of ArkWeb component.

The zoomAccess attribute controls whether the web content can be zoomed in or out using gestures.

Test Instructions:

  • When zoomAccess is false: Pinch gestures should not zoom the content
  • When zoomAccess is true: Pinch gestures should zoom the content normally
